# Uizard Review

**Uizard** creates professional app and website designs from hand-drawn sketches, screenshots, or text prompts. No design skills required.

## Key Features

- **Sketch-to-UI** — Convert wireframes into digital designs
- **Screenshot-to-Design** — Transform screenshots into editable UI
- **AI Theme Generation** — Generate complete design themes from text
- **Prototyping** — Clickable prototypes with interactivity
- **Component Library** — Thousands of pre-built UI components
- **Collaboration** — Real-time team editing and commenting
