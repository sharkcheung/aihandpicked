# Khroma Review

**Khroma** uses machine learning to understand your color preferences and generate palettes, gradients, and typography combinations you'll love.

## Key Features

- **Personalized Palettes** — Trains on preferences for colors you'll like
- **Infinite Combinations** — Generate endless color combinations
- **Multiple Formats** — Palettes, gradients, typography views
- **Accessibility Check** — Shows contrast ratios for accessibility
- **Color Codes** — Copy HEX, RGB, HSL, CSS values
- **Favorites** — Save and organize favorites
