# Sourcegraph Review

**Sourcegraph** is a universal code search and intelligence platform. It helps developers understand, navigate, and search across massive codebases with AI-powered features like Cody.

## Key Features

- **Code Search** — Search across thousands of repositories
- **Cody AI** — AI coding assistant integrated
- **Code Intelligence** — Jump to definitions, references
- **Batch Changes** — Large-scale code refactoring
- **Code Insights** — Track code health metrics
- **Self-Hosted** — Deploy on your infrastructure
