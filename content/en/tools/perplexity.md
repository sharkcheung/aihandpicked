# Perplexity AI Review

**Perplexity AI** is an AI-powered search engine that combines the best of web search with large language models. Unlike traditional search engines, it provides direct answers with citations, making research faster and more reliable.

## Key Features

- **Real-time web search** — Always up-to-date information
- **Cited sources** — Every answer includes links to original sources
- **Follow-up questions** — Conversational interface for deeper research
- **Pro mode** — Access to GPT-4 and Claude for enhanced answers
- **Collections** — Save and organize research threads
- **Mobile app** — Full-featured iOS and Android apps
