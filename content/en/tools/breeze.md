# Breeze Review
**Breeze** automates code review with AI. It reviews pull requests, detects bugs, suggests improvements, and integrates with GitHub.
## Key Features
- **Auto PR Review** — Reviews pull requests with AI feedback
- **Bug Detection** — Detects bugs and security issues
- **Code Suggestions** — Suggests improvements and best practices
- **GitHub Integration** — Works natively with GitHub
- **Multi-language** — Supports 20+ programming languages
- **Team Rules** — Enforces team coding standards
