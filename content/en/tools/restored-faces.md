# GFP-GAN Review

**GFP-GAN** (Generative Facial Prior GAN) is an open-source AI model for photo restoration. It excels at fixing old, damaged, or low-quality face photos.

## Key Features

- **Face restoration** — Repair damaged facial features
- **Enhancement** — Improve resolution and clarity
- **Color correction** — Fix faded or wrong colors
- **Open source** — Free to use and modify
- **Batch processing** — Handle multiple images
- **Local processing** — No cloud required
