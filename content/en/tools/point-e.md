# Point-E Review

**Point-E** is OpenAI's open-source 3D generation model that creates point clouds from text prompts. Unlike mesh-based generators, Point-E focuses on speed and accessibility for research applications.

## Key Features

- **Text-to-3D** — Generate 3D point clouds from descriptions
- **Open source** — Full code available on GitHub
- **Fast generation** — Results in seconds on GPU
- **Point cloud output** — Different from mesh-based approaches
- **Research-friendly** — Designed for experimentation
- **Customizable** — Modify and extend as needed
