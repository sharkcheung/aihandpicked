# AI Coding Assistants

AI coding tools help developers write code faster, debug issues, and learn new frameworks — directly in your IDE.

## Top AI Coding Tools

- **[GitHub Copilot]({{< relref "tools/github-copilot" >}})** — Industry standard, deeply integrated in VS Code
- **[Cursor]({{< relref "tools/cursor" >}})** — AI-first IDE built on VS Code
- **[Replit]({{< relref "tools/replit" >}})** — Online IDE with built-in AI agent
- **[Codeium]({{< relref "tools/codeium" >}})** — Free alternative with solid features

## What to Look For

- **IDE support** — VS Code, JetBrains, Vim, or browser-based?
- **Language coverage** — How many languages are supported?
- **Context awareness** — Can it understand your entire codebase?
- **Pricing** — Free tier available?
