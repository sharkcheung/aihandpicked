# Luma Dream Machine

Luma Dream Machine 是一款AI视频生成器，可从文字提示或图片创建逼真的视频。它是文字转视频领域的领先竞争者之一。

## 核心功能

- **文字转视频** — 从提示生成5秒片段
- **图片转视频** — 让静态图片动起来
- **相机运动** — 控制平移、缩放和移动
- **高真实感** — 照片级真实输出
- **快速生成** — 约2分钟出结果
- **延长视频** — 为片段增加更多秒数
