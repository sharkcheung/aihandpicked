# Masterpiece X

Masterpiece X 是一款AI驱动的3D角色生成器，专为VRChat、游戏和虚拟环境设计。它从文字提示创建完整绑定、动画就绪的角色。

## 核心功能

- **文字生成角色** — 从提示生成绑定的3D角色
- **自动绑定** — 角色自带骨骼用于动画
- **VRChat就绪** — 优化后可直接导入VRChat
- **风格选项** — 动漫、写实、风格化等
- **定制化** — 微调生成的角色
- **Mixamo兼容** — 支持Mixamo动画
