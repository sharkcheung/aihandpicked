# GitHub Copilot

GitHub Copilot 是使用最广泛的AI编程助手，在你输入时提供实时代码建议。它由OpenAI模型驱动，直接集成到VS Code、JetBrains和其他流行的IDE中。

## 核心功能

- **实时建议** — 编写代码时AI自动补全
- **多语言支持** — 支持Python、JavaScript、TypeScript等
- **上下文感知** — 理解你的整个代码库
- **聊天界面** — 询问关于代码的问题
- **测试生成** — 自动生成单元测试
- **代码审查** — AI驱动的PR审查
