# Attio 评测

**Attio** 是一款专为现代快速团队设计的AI原生CRM。它将智能自动化与关系智能相结合。

## 核心功能

- **AI数据录入** — AI-Powered Data Entry
- **智能工作流** — Smart Workflows
- **关系图谱** — Relationship Mapping
- **灵活数据模型** — Flexible Data Model
- **实时同步** — Real-time Sync
- **AI活动摘要** — AI Activity Summaries
