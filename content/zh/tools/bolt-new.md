# Bolt 评测

**Bolt** 由StackBlitz开发，让您完全在浏览器中通过提示、运行、编辑和部署全栈Web应用。

## 核心功能

- **全栈生成** — Full-Stack Generation
- **浏览器运行时** — In-Browser Runtime
- **AI代码编辑器** — AI Code Editor
- **即时预览** — Instant Preview
- **NPM生态** — NPM Ecosystem
- **一键部署** — One-Click Deploy

## 定价

| 方案 | 价格 | 核心功能 |
|------|------|----------|
| 有限的生成次数 | $0 | Limited generations |

## 优势

✅ 几分钟内从想法到部署

✅ 不需要本地开发环境

✅ 支持全栈——不仅是前端

✅ 非常适合快速原型和MVP

## 劣势

❌ 生成代码可能较乱

❌ 调试AI代码可能很沮丧

快速原型设计的开发者、构建MVP的非技术创始人，以及无需复杂设置创建Web应用的人。
