# OpenAI Whisper

Whisper 是OpenAI的开源自动语音识别（ASR）模型。它能以惊人的准确度将音频转录为文字，支持100多种语言，包括自动翻译功能。

## 核心功能

- **多语言支持** — 100多种语言
- **翻译** — 将语音转换为英文文本
- **开源** — 在你的硬件上本地运行
- **高准确度** — 接近人类的转录质量
- **多种大小** — 从tiny（快速）到large（准确）
- **时间戳** — 字幕用的字级时间标记
