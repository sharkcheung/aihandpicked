# Breeze 评测
**Breeze** 用AI自动化代码审查。审查PR、检测bug、建议改进，与GitHub集成。
## 核心功能
- **自动PR审查** — Auto PR Review
- **Bug检测** — Bug Detection
- **代码建议** — Code Suggestions
- **GitHub集成** — GitHub Integration
- **多语言** — Multi-language
- **团队规则** — Team Rules

## 定价

| 方案 | 价格 | 核心功能 |
|------|------|----------|
| Free | $0 | Open source, limited reviews |
| Pro | $15/mo | Unlimited reviews, full features |

## 优势

✅ 快速自动代码审查

✅ 轻松捕获常见bug

✅ 易于设置

✅ 好的GitHub集成
