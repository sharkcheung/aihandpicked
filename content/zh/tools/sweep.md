# Sweep 评测

**Sweep** 将GitHub Issues转换为Pull Requests。给它一个任务，它会自动规划、编写、测试并创建PR。

## 核心功能

- **Issue转PR** — Issue-to-PR
- **多文件更改** — Multi-file Changes
- **测试生成** — Test Generation
- **规划与执行** — Plan & Execute
- **上下文感知** — Context Awareness
- **GitHub原生** — GitHub Native

✅ 有效自动化常规编码任务

✅ 理解代码库上下文

✅ 在重复任务上节省时间
