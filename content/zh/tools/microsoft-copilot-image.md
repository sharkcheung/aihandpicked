# Microsoft Copilot

Microsoft Copilot 是一款免费的AI助手，由GPT-4和DALL-E 3驱动。它回答问题、生成图片、编写内容，并与Microsoft 365应用集成。

## 核心功能

- **GPT-4驱动** — 免费使用最新OpenAI模型
- **DALL-E 3图片** — 聊天内生成图片
- **网络访问** — 搜索互联网获取最新信息
- **Microsoft 365集成** — 与Word、Excel等协同
- **语音模式** — 手机上与Copilot对话
- **多种模式** — 创意、平衡、精准
