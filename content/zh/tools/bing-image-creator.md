# Bing Image Creator

Bing Image Creator 是微软免费的AI图片生成器，由DALL-E 3驱动。它是创建AI图片的最佳免费选择之一，质量惊人且无需订阅。

## 核心功能

- **DALL-E 3驱动** — 与付费DALL-E相同质量
- **免费使用** — 每天15次加速，之后变慢
- **简单提示** — 简单自然语言输入
- **多输出** — 每次提示生成4张图片
- **移动应用** — iOS和Android可用
- **Copilot集成** — 通过聊天生成
