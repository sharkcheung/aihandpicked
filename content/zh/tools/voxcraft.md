# VoxCraft

VoxCraft是专为社交媒体创作者打造的AI视频生成平台，专注于热门趋势和病毒式内容，让创作者能够快速制作出易于传播的短视频。

## 核心功能

- **热门模板** — 随时可用的爆款视频模板
- **文字生成视频** — 从描述生成视频
- **自动比例适配** — 针对各平台优化
- **音乐同步** — AI自动匹配音乐和画面
- **特效库** — 流行特效和转场效果
- **快速导出** — 渲染速度快
