# Figma AI Review

**Figma AI** integrates artificial intelligence directly into Figma, the industry-standard design tool. It helps designers work faster by automating layout tasks, generating content, and suggesting improvements.

## Key Features

- **Auto-layout suggestions** — AI recommends optimal spacing and alignment
- **Content generation** — Creates realistic placeholder text and images
- **Component creation** — Generate buttons, cards, and UI elements from text prompts
- **Design to code** — Convert designs to production-ready code
- **Accessibility checks** — AI-powered contrast and readability analysis
