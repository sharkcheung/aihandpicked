# Gamma Review

**Gamma** is an AI-powered platform for creating presentations, documents, and webpages. It generates content from prompts and offers a modern, web-native approach to visual content creation.

## Key Features

- **AI generation** — Create decks and docs from prompts
- **Modern templates** — Web-native, responsive designs
- **Interactive elements** — Embed videos, GIFs, and apps
- **Collaboration** — Real-time editing with comments
- **Analytics** — Track engagement on shared content
- **Export options** — PDF, PPT, and web publishing
