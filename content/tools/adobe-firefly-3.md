# Adobe Firefly 3 Review

**Adobe Firefly 3** is Adobe's generative AI suite integrated into Creative Cloud. It generates images, vectors, and effects directly in Photoshop, Illustrator, and Premiere.

## Key Features

- **Text to Image** — Create images from descriptions
- **Generative Fill** — Extend or modify images in Photoshop
- **Vector Generation** — Create editable vectors
- **Text Effects** — Apply styles to typography
- **Video Generation** — Generate video clips (beta)
- **Commercial Safe** — Trained on licensed content

## Pricing

| Plan | Price | Key Features |
