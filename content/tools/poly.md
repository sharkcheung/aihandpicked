# Poly Review

**Poly** creates photorealistic PBR textures from text or images for games, AR/VR, and 3D design.

## Key Features

- **Text-to-Texture** — Generate PBR textures from text
- **Image-to-Texture** — Create textures from reference photos
- **PBR Materials** — Complete PBR material sets
- **3D Capture** — Scan real objects with your phone
- **Seamless Textures** — Perfectly tileable textures
- **Export Formats** — OBJ, GLTF, USDZ, FBX
