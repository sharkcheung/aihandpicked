# Masterpiece X Review

**Masterpiece X** is an AI-powered 3D character generator specifically designed for VRChat, games, and virtual environments. It creates fully rigged, animation-ready characters from text prompts.

## Key Features

- **Text-to-character** — Generate rigged 3D characters from prompts
- **Auto-rigging** — Characters come with skeletons for animation
- **VRChat-ready** — Optimized for direct import to VRChat
- **Style options** — Anime, realistic, stylized, and more
- **Customization** — Fine-tune generated characters
- **Mixamo support** — Compatible with Mixamo animations
