# Cursor Review

**Cursor** is a fork of VS Code that's been reimagined for the AI era. It's not just an AI plugin — it's an entirely new coding experience where AI is woven into every interaction.

## Key Features

- **Tab Complete** — AI-powered autocomplete that understands your codebase
- **Cmd+K (Edit)** — Describe changes in plain language, Cursor applies them
- **Cmd+L (Chat)** — Context-aware chat that sees your entire project
- **Composer** — Multi-file edits across your entire codebase
- **Codebase Indexing** — AI understands your full project structure
- **@ Mentions** — Reference specific files, docs, or code in prompts

## Who It's Best For
