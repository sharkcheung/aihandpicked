# Sketch AI Review

**Sketch AI** adds intelligent features to Sketch, the pioneering Mac design tool. It helps designers generate placeholder content, automate repetitive tasks, and create assets more efficiently.

## Key Features

- **Smart content** — Generate realistic placeholder text and images
- **Auto-naming** — AI names layers and groups intelligently
- **Asset generation** — Create icons and illustrations from prompts
- **Layout suggestions** — AI recommends spacing and alignment
- **Design linting** — Check for consistency issues automatically
