# Bito Review

**Bito** is an AI coding assistant that integrates GPT-4 into your development workflow. It helps write code, generate unit tests, explain code, and improve code quality.

## Key Features

- **Code Generation** — Write code from natural language
- **Unit Test Generation** — Auto-generate comprehensive tests
- **Code Explanation** — Understand complex codebases
- **Code Review** — AI-powered code review suggestions
- **Multiple LLMs** — Choose between GPT-4, Claude, and more
- **IDE Integration** — VS Code, JetBrains extensions
